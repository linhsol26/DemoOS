Demo for Operating System's subject.
4 algorithms (FCFS, SJF, SRTF, RR) with multi I/O.

Example:

| Arrive | CPU | I/O  | CPU  |
| ------------- | ------------- | ------------- | ------------- |
|  0 | 2  | 3  | 2  |
|  1  | 1  | 3  | 4  |
|  2  | 3  | 5  | 7  |

Then the input gonna be:

| Arrive | CPU | I/O  |
| ------------- | ------------- | ------------- |
|  0 | 2, 2  | 3  |
|  1  | 1, 4  | 3  |
|  2  | 3, 7  | 5  |

Please check the input before you run.

Also available for different CPU and I/O of each process.
